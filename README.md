## Dependencias
* _**Composer**_
* _**PHP >= 7**_

## Instalación
* **git clone _[link]_**
* **composer update** (Actualizar dependencias)
* **php artisan key:generate** (Encriptación)
* **php artisan serve** (Servidor)
